About this Project

This website is part of my personal portfolio showcasing my skills in cloud development. The project is built using AWS services, including EC2 for hosting, S3 for static storage, and Route 53 for DNS management. It's designed to be simple, responsive, and demonstrate best practices in cloud infrastructure.

  • AWS EC2 – for hosting the application/backend file
  • AWS S3 – for static file storage
  • AWS Route 53 – for DNS management
  • AWS CloudFront – for the rapid delivery of content thanks to CDN
  • AWS RDS – for managing a MySQL relational database
  • AWS SNS – for sending notification about new messages from contact form
  • HTML & CSS – for front-end design
  • GitHub – for version control and project management

Weather Project

This is my second portfolio project, showcasing my knowledge in cloud computing and web development. The application provides daily weather updates for Wrocław and highlights the city's attractions and events. This project demonstrates my ability to design and implement cloud-based solutions using AWS services for Wrocław.

  • AWS Lambda – for fetching and processing weather data
  • AWS S3 – for storin raw weather data in JSON format
  • AWS DynamoDB – for saving processed weather data for quick access
  • AWS EventBridge – for automating daily weather data fetching
  • API Gateway – for exposing processed data via REST API
  • AWS Amplify – for hosting and deploying frontend application
  • OpenWeatherMap API – for providing real-time weather data
  • GitHub – for version control and project management
